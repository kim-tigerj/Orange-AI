# q1 (Qwen Mastery 1) 실험 가이드

## 🔬 연구 목표
- **상용 AI 대체**: Qwen2.5-Coder 32B를 통해 Claude Code 및 Gemini API를 완전히 대체.
- **역할군 각성**: Qwen 모델이 에코시스템 내 핵심 역할(정팀장, 오감독, 팀원)을 완벽히 수행하도록 최적화.
- **M4 하드웨어 가속**: 128GB 통합 메모리를 활용, 고정밀(Q8_0 이상) 가중치로 상용 AI 수준의 지능 확보.

## 📝 연구 지침
- 모든 실험 데이터는 정량적으로 기록하며, '상용 AI 대비 성능 %'를 반드시 포함한다.
- Qwen 모델의 '도구 호출(Tool Calling)' 정확도를 최우선으로 검증한다.
