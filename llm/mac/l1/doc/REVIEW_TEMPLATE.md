# l1 4축 리뷰 시트 (Supervisor 전용)

이 문서는 오감독(Supervisor)이 l1(Mac/LLM) 연구 결과를 검증할 때 사용하는 표준 서식입니다.

## 📋 검증 항목

### 1. 지능 (Functionality)
- [ ] 로컬 모델의 답변이 정확하고 신뢰할 수 있는가?
- [ ] 도구 호출(Tool Use)의 성공률이 실사용 가능한 수준인가?

### 2. 구현의 정도 (Quality)
- [ ] 프롬프트 최적화 및 파라미터 세팅이 정교한가?
- [ ] 실험 데이터가 객관적이고 재현 가능한가?

### 3. 타 기능과의 관계 (Consistency)
- [ ] 윈도우 앱의 백엔드 규격과 호환되는가?

### 4. 시스템 영향도 (System Impact)
- [ ] CPU/RAM 점유율이 '경량화' 비전에 부합하는가?
- [ ] **[New]** 불필요한 API 호출 없이 최소한의 비용으로 작업을 완료했는가?
