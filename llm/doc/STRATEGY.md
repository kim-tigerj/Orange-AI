# The Brain: Mac LLM Research Strategy (두뇌 연구소)

## 🎯 존재 이유
Orange Code의 지능(Intelligence)을 담당하는 연구 기지입니다. M4 맥북프로의 강력한 하드웨어를 활용하여 외부 API 의존도를 낮추고 로컬에서 최상의 지능을 배양하는 것을 목표로 합니다.

## ⚙️ 연구 전략
- **로컬 최적화**: M4 통합 메모리와 GPU(Metal)를 극한으로 활용하는 추론 경로 연구.
- **도구 호출 정밀화**: 로컬 모델이 윈도우 앱의 복잡한 도구들을 실수 없이 호출하도록 프롬프트 및 파인튜닝 가이드 수립.
- **비용 0화**: 유료 API 호출 없이도 업무 생산성을 유지할 수 있는 로컬 에코시스템 구축.
