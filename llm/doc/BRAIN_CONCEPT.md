# The Brain: Research Philosophy (연구부 전체 개념)

## 🏛️ 상위 거버넌스
이 문서는 [GOVERNANCE.md](../../doc/GOVERNANCE.md)의 비전을 지능 연구 측면에서 구체화한 문서입니다.

## 🔬 연구 원칙
- **지능의 배양**: 로컬 리소스를 활용하여 외부 API 없이도 최고 수준의 논리력을 확보한다.
- **최적의 가성비**: 모델 파라미터 크기와 추론 속도의 최적 지점(Sweet Spot)을 끊임없이 탐색한다.
- **특수 지능 확보**: 일반적인 대화가 아닌, Orange Ecosystem의 도구(Tools)를 가장 잘 다루는 특화된 지능을 만든다.
