# Orange Ecosystem Glossary (용어집)

이 문서는 에코시스템 내에서 사용되는 핵심 용어들을 정의합니다. 모든 인스턴스는 대화 및 문서 작성 시 이 정의를 따릅니다.

## 🏛️ 핵심 역할 및 주체
- **Head (대표님)**: 시스템의 정점이자 최종 의사결정권자. (인간)
- **General Manager (GM/정팀장)**: 자원 배분 및 공정을 관리하는 실무 사령탑. (AI)
- **Supervisor (오감독)**: 4축 리뷰를 통해 품질을 검증하는 감시자. (AI)
- **Member (팀원)**: 단위 작업을 수행하는 전문 일꾼. (AI)

## 🏗️ 구조 및 플랫폼
- **The Body (몸)**: 윈도우 네이티브 앱 구현부 (code/windows).
- **The Brain (머리)**: 로컬 LLM 지능 연구부 (llm/mac).
- **wN (Windows Version N)**: 윈도우 앱의 특정 발전 단계 (예: w1).
- **lN (Local LLM Version N)**: 지능 연구의 특정 발전 단계 (예: l1).

## ⚙️ 프로세스 및 기술
- **4축 리뷰**: 기능/구현/관계/영향도를 기준으로 하는 품질 검증 프로세스.
- **Context Pumping**: 계층형 문서를 따라 필요한 지식을 AI에게 주입하는 기법.
- **Intelligence Bridge**: 맥북의 연구 결과가 윈도우 앱의 실무 지침으로 변하는 가교.
