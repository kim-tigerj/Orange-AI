# The Body: Windows Platform Strategy (윈도우 지부)

## 🎯 존재 이유
Orange Code의 실체(Entity)이자 사용자와 만나는 최전선입니다. Win32/Direct2D 네이티브 기술을 통해 가장 가볍고 빠른 사용자 경험을 제공하는 것을 목표로 합니다.

## ⚙️ 플랫폼 전략
- **초경량/초고속**: Electron 등의 무거운 프레임워크를 지양하고 순수 네이티브의 성능을 극대화함.
- **다중 인스턴스**: 자원 효율성을 바탕으로 수십 개의 정팀장 세션을 동시에 구동할 수 있는 구조 확보.
- **실무 중심**: 사용자의 워크플로우를 방해하지 않는 직관적인 인터페이스와 도구 제공.
