# w1 (Windows Version 1) 상세 명세

## 🛠️ 작업 목표
- 기존 OrangeCode 프로젝트의 핵심 기능을 안정적으로 이식함.
- Win32/Direct2D 기반의 마크다운 렌더링 엔진 완성.
- l1 연구소와 연동 가능한 로컬 백엔드 인터페이스 구축.

## 📝 실무 지침
- 이사는 단계적으로 진행하며, 각 단계마다 빌드 무결성을 확인한다.
- 모든 결정 사항은 이 doc 폴더 내의 작업 로그에 기록한다.
